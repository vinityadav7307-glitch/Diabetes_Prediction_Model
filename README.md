# Diabetes Prediction using Machine Learning

This project predicts whether a person is diabetic or not using machine learning techniques.

## Dataset
PIMA Indians Diabetes Dataset  
Source: https://www.kaggle.com/datasets/uciml/pima-indians-diabetes-database  

The dataset file (diabetes.csv) is included in this repository.


## Features Used
- Pregnancies
- Glucose
- BloodPressure
- SkinThickness
- Insulin
- BMI
- DiabetesPedigreeFunction
- Age

## Model Used
Support Vector Machine (SVM)

## Libraries
- numpy
- pandas
- scikit-learn

## Steps
1. Data loading and preprocessing
2. Feature scaling using StandardScaler
3. Train-test split
4. Model training using SVM
5. Model evaluation
6. Prediction system

## How to Run
1. Install required libraries
2. Open the notebook
3. Run all cells in order

## Output
Model predicts whether the person is diabetic or not.
